常见问题
接入前最常被问到的几件事。还有疑问?进 Discord 直接问。
接入要改多少代码?+
两行。把 api_key 换成 GeniusCode 的，base_url 指向 https://api.geniuscode.online/v1。OpenAI SDK、流式、工具调用全部不用动，业务代码零改写。
路由是怎么决定用哪个模型的?+
按你配置的任务类型、预算上限和上游实时可用性综合打分选优。每次响应头都会带 x-gc-picked(这次选了谁)和 x-gc-cost(花了多少)，决策可审计、可复盘。你也可以手动指定模型，绕过自动路由。
比直连上游贵吗?+
按 token 实时计费，价格表完全公开，所有模型单价一目了然。无月费、不绑套餐，用多少付多少。注册即送体验额度，无需绑卡。
上游模型挂了会怎样?+
自动切换到同任务的备用模型，单次请求不漏单，调用方无感知。备用链可在控制台按任务自定义，谁主谁备由你说了算。
我的数据安全吗?+
我们不存储对话正文，仅保留 token 计数与路由元数据用于计费和审计。企业版支持自带模型额度与私有化部署。
支持哪些模型?+
已接入 DeepSeek、通义千问、Kimi、GLM、MiniMax 等 30+ 主流模型（含推理、对话、代码专用）。新增模型即接即用，不用改你这边任何代码。完整清单见模型 & 价格。

常见错误码

  • 401 key 无效或已停用 — 检查 Authorization 头
  • 402 余额不足 — 去 开通会员
  • 429 触发上游限流 — 已自动重试 / 降级，仍失败请稍后再试
  • 503 上游不可用 — 路由已尝试故障转移，可手动指定备用模型

隐私政策

最近更新：2026 年

1. 我们收集什么

账户信息（邮箱）、用于计费与审计的调用元数据（模型、token 计数、时间、状态、成本）。我们不存储你的 prompt / 模型回复正文。

2. 如何使用

仅用于提供服务、计费对账、故障排查与安全防护，不向第三方出售你的数据。请求会转发至上游模型供应商以完成推理。

3. 数据安全

传输全程 HTTPS 加密；API Key 等密钥仅你可见，请妥善保管。如发生密钥泄露，请尽快在控制台重置。

4. 你的权利

你可随时导出用量记录、注销账户。注销后我们将在合理期限内删除可识别个人身份的数据（法律法规要求保留的除外）。

5. 联系

隐私相关问题请邮件 privacy@geniuscode.online。

使用条款

最近更新：2026 年

1. 服务说明

GeniusCode 提供多模型 API 统一中转服务，以 OpenAI 兼容接口转发至上游供应商。我们按可用性尽力保障服务，但不对上游模型输出的准确性负责。

2. 可接受使用

禁止用于违法违规、侵犯他人权益、生成恶意/欺诈内容、攻击或滥用接口（如绕过限额、压力测试未授权目标）等用途。一经发现，我们有权暂停或终止账户且不予退款。

3. 账户与额度

你需对账户下所有调用负责，包括 API Key 泄露导致的消费。额度按 token 实时计费，全模型通用、自购买起 12 个月内有效。

4. 服务变更与中止

我们可能因上游调整、维护或不可抗力变更或暂停部分模型/功能，并尽量提前告知。

5. 责任限制

在法律允许范围内，因使用本服务产生的间接损失我们不承担责任；总责任以你最近 3 个月支付金额为上限。

6. 联系

条款相关问题请邮件 contact@geniuscode.online。

退款政策

1. 预付额度

会员开通 / 续充均为预付额度，按实际 token 用量扣费。

自购买之日起 7 天内、且未发生任何调用的未使用额度，可申请全额退款。已产生调用的，按未消费余额比例评估退款。
