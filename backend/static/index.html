常见问题 / FAQ
接入前最常被问到的几件事。还有疑问?进 Discord 直接问。
接入要改多少代码?+
两行。把 api_key 换成 GeniusCode 的，base_url 指向 https://api.geniuscode.online/v1。OpenAI SDK、streaming、tool calling 全部不用动，业务代码零改写。
路由是怎么决定用哪个模型的?+
按你配置的任务类型、预算上限和上游实时可用性综合打分选优。每次响应头都会带 x-gc-picked(这次选了谁)和 x-gc-cost-usd(花了多少)，决策可审计、可复盘。你也可以手动指定模型，绕过自动路由。
比直连上游贵吗?+
透传上游原价 ×1.6，覆盖路由智能、服务器和故障转移成本。价格表完全公开，含上游原价对照——你随时能算清楚加价加在哪。注册即送 $5 路由额度，无需绑卡。
上游模型挂了会怎样?+
自动 fallback 到同任务的备用模型，单次请求不漏单，调用方无感知。fallback 链可在控制台按任务自定义，谁主谁备由你说了算。
我的数据安全吗?+
我们不存储 prompt / completion 正文，仅保留 token 计数与路由元数据用于计费和审计。企业版支持 BYOK(自带上游 key)和私有化部署。
支持哪些模型?+
当前已接入 OpenAI 全系 + DeepSeek 全系，Anthropic 接入中(V2)。新增模型即接即用，不用改你这边任何代码。

常见错误码

  • 401 key 无效或已停用 — 检查 Authorization 头
  • 402 余额不足 — 去 充值
  • 429 触发上游限流 — 已自动重试 / 降级，仍失败请稍后再试
  • 503 上游不可用 — 路由已尝试 fallback，可手动指定备用模型
